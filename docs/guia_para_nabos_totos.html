AI Macro Market Intelligence Dashboard

A very simple guide to understand what this project does, why it exists, how to run it, and how to read the dashboard without pretending to be a Wall Street wizard.

This is an educational analytics project. It is not financial advice. Do not buy or sell anything only because a chart looks pretty.

1. What Is This Project?

This project downloads market data for companies, ETFs and commodities related to artificial intelligence, semiconductors, precious metals, energy and transport/logistics.

Then it cleans the data, calculates risk and return metrics, groups similar assets, detects big market events, and creates forecast scenarios for the next 1 to 7 years.

2. The Simple Idea

Download

Get historical prices from yfinance.

Store raw data

Save the original download in SQLite so we can audit it later.

Clean

Remove broken rows, bad prices and duplicates.

Analyse

Load clean data into DuckDB and build charts, rankings and forecasts.

3. Data Flow

yfinance
SQLite staging
Quality checks
DuckDB analytics
Streamlit dashboard

The important rule: Streamlit reads from DuckDB. It does not directly read raw yfinance downloads. This makes the project much closer to a real data platform.

4. What Each Database Does

5. How To Run It

cd AI_Macro_Market_Intelligence_Dashboard
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py

Then open the local URL shown by Streamlit, usually http://localhost:8501.

6. First Thing To Click

In the sidebar, choose a few tickers and click Refresh market data. This creates a new download batch, downloads prices, stores raw data in SQLite, transforms it, and rebuilds DuckDB.

If one ticker fails, the app should continue. Failed tickers are logged and shown to the user.

9. How To Read The Charts

Cumulative return

If the line goes up, the asset performed well over that period. If it falls hard, check drawdown and volatility.

Risk-return scatter

Up means more return. Right means more risk. Top-right can be exciting but dangerous.

Correlation heatmap

Dark positive values mean assets move together. Negative values can help diversification.

Forecast band

The middle line is a scenario. The band shows uncertainty. Wide band means low confidence.

10. What Is Clustering?

Clustering groups assets that behave similarly. For example, some assets may be high-growth and high-volatility, while others may be defensive or behave like commodity hedges.

Clustering does not know the future. It only groups assets using the features we give it.

11. What Are Forecasts?

Forecasts are educated scenarios based on historical data. This project includes baseline methods, exponential smoothing, ARIMA, random forest and Monte Carlo simulation.

The right way to read a forecast is: "if the historical behaviour continues in this style, this range of outcomes could happen."

Markets are messy. Data can be wrong. Tickers can fail. Forecasts can fail. This project is useful because it makes the assumptions visible instead of hiding them behind one shiny number.
